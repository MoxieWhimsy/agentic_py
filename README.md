# agentic py project

Based on the ***Build an AI Agent in Python*** course on Boot-dot-dev (https://www.boot.dev/courses/build-ai-agent-python)

## How to use
**DO NOT USE IN PRODUCTION**. While it has some limitations built-in, it also allows arbitrary Python code to run.

Pass a prompt as an argument to the main.py. Optionally pass `--verbose` to get a lot more output than usual.

`uv run main.py "type-a-prompt-here" [--verbose]`
